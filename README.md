The hangman is a famous game usually played on paper with drawing a hanging man on every mistake.
My idea was to evolve with time and also shift this game onto a digital platform 
The game contains of multiple levels and also AI assistance on the hard level so the user can have a chance to succeed
The backend of the game is attached with a dictionary from which words are selected and everytime are presented to be guessed
The test.py file was to test the stage of game its a good practice so at initial stage you can debug
